# 輿情監控系統

**目前活躍：否**
**建立日期：2026-05-08**
**目標：** 監控 Dcard、PTT、Google News、Threads 的醫美/醫療輿情，推播 LINE 通知，提供公司共用網站 Dashboard

---

## 架構

- **後端**：FastAPI on Render.com（獨立新服務）
- **資料庫**：Supabase（免費 PostgreSQL + Auth）
- **前端**：靜態 HTML on Surge.sh
- **觸發**：cron-job.org 每 15 分鐘 → `/sentiment-scan`
- **費用**：$0

## 資料來源
- Dcard：非官方 API（不需申請）
- Google News：RSS
- PTT：公開 API
- Threads：Firecrawl 爬公開搜尋頁（官方 API 待手機驗證解決）

## Supabase 資料表
- `articles`：抓到的文章（source, title, url, category, published_at）
- `article_actions`：用戶標記（read/processing/ignored + 備注）
- `user_settings`：LINE User ID + 通知偏好
